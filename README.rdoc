About The Application

It's an application that allows a user to view details of Countries, States inside each country, and Cities inside each state.

Each place has parameters like population, name, wikilink, area.

Population of cities add up to population of a state, states add up to a country.

Every Country has additional parameters like currency, time zone.

Country and state have an additional parameer, "literacy".
  
Ruby Version: 2.1.5

Rails Version: 4.2.0

Database: SQLite
